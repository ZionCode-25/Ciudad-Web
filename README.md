## 🎯 Descripción

**Ciudad Web** es un juego educativo interactivo diseñado para enseñar HTML y CSS de manera práctica y divertida. Los usuarios aprenden construyendo una ciudad web paso a paso, completando 18 niveles organizados en categorías que van desde conceptos básicos hasta técnicas avanzadas.

### 🎓 Objetivos Educativos

- **Aprendizaje Práctico**: Escribir código real desde el primer nivel
- **Progresión Gradual**: Conceptos organizados de básico a avanzado
- **Feedback Inmediato**: Validación automática y vista previa en tiempo real
- **Gamificación**: Sistema de progreso y logros para mantener la motivación

## 🎮 Cómo Jugar

### 📚 **Categorías de Niveles**

#### 🏗️ **HTML Básico** (8 niveles)

1. **Primer ladrillo** - Elementos básicos `<h1>` y `<p>`
2. **Imagen en la fachada** - Etiqueta `<img>` con atributos
3. **Navegación** - Enlaces con `<a href>`
4. **Lista de compras** - Listas `<ul>` y `<li>`
5. **Formulario simple** - Inputs y botones
6. **Tabla básica** - Estructura de tablas
7. **Estructura semántica** - HTML5 semántico
8. **Atributos avanzados** - Validación de formularios

#### 🎨 **CSS Estilos** (10 niveles)

1. **Color en el título** - Propiedades de color
2. **Box model** - Padding, border, margin
3. **Flexbox básico** - Layout flexible
4. **Grid básico** - Sistema de cuadrícula
5. **Pseudo-clases** - Estados hover, focus
6. **Transiciones** - Animaciones suaves
7. **Responsive simple** - Media queries
8. **Animación CSS** - Keyframes y animations
9. **Variables CSS** - Custom properties
10. **Posicionamiento** - Position absolute, relative

### 🎯 **Mecánica del Juego**

1. **Lee las instrucciones** del nivel actual
2. **Escribe tu código** en el editor integrado
3. **Ve la vista previa** en tiempo real
4. **Haz clic en "Comprobar"** para validar tu solución
5. **Avanza al siguiente nivel** una vez completado
6. **Usa el botón de ayuda** si necesitas orientación
