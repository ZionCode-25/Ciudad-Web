## 🎯 Descripción

**Ciudad Web** es un juego educativo interactivo diseñado para enseñar HTML y CSS de manera práctica y divertida. Los usuarios aprenden construyendo una ciudad web paso a paso, completando 18 niveles organizados en categorías que van desde conceptos básicos hasta técnicas avanzadas.

- **Aprendizaje Práctico**: Escribir código real desde el primer nivel

- **Feedback Inmediato**: Validación automática y vista previa en tiempo real

## 🧠 Conceptos Fundamentales

### 🏗️ **HTML: El Esqueleto de la Web**

HTML (HyperText Markup Language) es el lenguaje de marcado que define la estructura y contenido de las páginas web.

#### **Conceptos Clave:**

- **Elementos y Etiquetas**: Bloques de construcción básicos
- **Atributos**: Propiedades que modifican el comportamiento
- **Semántica**: Dar significado al contenido para máquinas y humanos
- **Accesibilidad**: Hacer el contenido usable para todos
- **SEO**: Optimización para motores de búsqueda

#### **Ejemplo Práctico:**

```html
<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8" />
    <title>Mi Primera Página</title>
  </head>
  <body>
    <header>
      <h1>Bienvenido a mi sitio</h1>
      <nav>
        <a href="#inicio">Inicio</a>
        <a href="#sobre-mi">Sobre mí</a>
      </nav>
    </header>
    <main>
      <section id="inicio">
        <h2>Página de Inicio</h2>
        <p>Este es mi primer sitio web.</p>
      </section>
    </main>
  </body>
</html>
```

### 🎨 **CSS: El Diseño de la Web**

CSS (Cascading Style Sheets) controla la presentación visual de los documentos HTML.

#### **Conceptos Clave:**

- **Selectores**: Cómo apuntar a elementos específicos
- **Cascada**: Cómo se aplican y priorizan los estilos
- **Box Model**: El modelo de cajas que define el layout
- **Flexbox y Grid**: Sistemas modernos de layout
- **Responsive Design**: Adaptación a diferentes pantallas

#### **Ejemplo Práctico:**

```css
/* Estilos globales */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

/* Layout con Flexbox */
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 20px;
}

/* Responsive Design */
@media (max-width: 768px) {
  .container {
    flex-direction: column;
    text-align: center;
  }
}
```

### 🔄 **Integración HTML + CSS**

La magia ocurre cuando HTML y CSS trabajan juntos:

1. **HTML proporciona la estructura** (qué es cada cosa)
2. **CSS proporciona la presentación** (cómo se ve)
3. **Juntos crean experiencias web completas**
